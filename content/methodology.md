# Methodology

1. **Ingest** — episode transcripts pulled via `yt-dlp` (English captions), frozen with SHA-256. *Transcripts are kept private (copyright); only derived synthesis is published.*
2. **Synthesize** — per-section canonical pages: deterministic protocol/passage extraction, then LLM synthesis into confidence-graded findings. Citations preserved.
3. **Deep research** — 12 contested topics researched by two independent agents (web + primary literature), then **reconciled** (agreements raise confidence; conflicts flagged).
4. **Audit** — adversarial fact-check per section; citations verified by fetching them; corrections applied.

**Confidence tags:** **[high]** = strong replicated human evidence · **[medium]** = emerging/mixed · **[low]** = speculative / preclinical-only / creator-opinion.

**Limits:** synthesis can still contain errors; creator framing is flagged but not exhaustively. This is a starting map, not a substitute for primary sources or professional advice.
