## Key researchers

| Researcher | Field |
|---|---|
| Eric R. Kandel | Molecular and cellular neuroscience of memory; synaptic plasticity (Nobel laureate) |
| Karl J. Friston | Computational neuroscience; functional neuroimaging, free-energy principle and predictive coding |
| Stanislas Dehaene | Cognitive neuroscience of consciousness, reading, numerical cognition |
| Edvard Moser and May-Britt Moser | Spatial navigation; grid cells and the entorhinal-hippocampal system (Nobel laureates) |
| Daniel Kahneman | Cognitive psychology; judgment, decision-making and dual-process theory |
| Antonio Damasio | Neuroscience of emotion, consciousness and decision-making (somatic marker hypothesis) |
| Matthew Walker | Sleep neuroscience; sleep, memory consolidation and health |
| Joseph E. LeDoux | Neuroscience of emotion, fear and the amygdala |
| Nancy Kanwisher | Cognitive neuroscience of vision; functional specialization of cortex (face/place areas) |
| Earl K. Miller | Prefrontal cortex; working memory, cognitive control and executive function |
| Wolfram Schultz | Reward prediction error and dopamine signaling in decision-making |
| Michael Gazzaniga | Cognitive neuroscience; split-brain research and hemispheric specialization |
| Elizabeth Loftus | Cognitive psychology of memory; false memory and eyewitness reliability |
| Larry Squire | Neuropsychology of memory systems; declarative vs. nondeclarative memory |
| Giulio Tononi | Theoretical neuroscience of consciousness (integrated information theory) and sleep |
