# Contributing

This is an evidence-graded knowledge base. The bar is **correctness + calibration**, not volume.

## Good contributions
- **Fix a miscited or wrong claim** — cite the primary source (DOI/PubMed preferred).
- **Recalibrate a confidence tag** — explain why `[high]/[medium]/[low]` should change.
- **Flag creator framing that outruns evidence**, with a source.
- **Add a contested-claims note** where the literature genuinely disagrees.

## Rules
- Cite primary/peer-reviewed sources for factual claims; link them inline.
- Keep confidence tags honest: `[high]` = strong replicated human evidence only.
- Don't paste copyrighted transcript text. Summarize + link.
- One topic per PR; quote the exact line you're changing.

## Confidence tags
`[high]` strong replicated human evidence · `[medium]` emerging/mixed · `[low]` speculative / preclinical-only / creator-opinion.
